# Plano de Testes de Usabilidade

## 1. Cenários e Tarefas (O que será testado)

| Cenário de Teste | Objetivo | Tarefas | Resultado Esperado |

| **Cadastro de Usuário** | Verificar se o usuário consegue realizar o cadastro | Acessar tela de cadastro, preencher os dados e finalizar o cadastro | O usuário deve concluir o cadastro sem dificuldades |
| **Login no Sistema** | Verificar se o usuário consegue acessar o sistema | Inserir e-mail e senha e selecionar “Entrar” | O usuário deve acessar o sistema de forma simples |
| **Busca de Assistências Técnicas** | Verificar se o usuário consegue localizar assistências próximas | Acessar tela inicial, visualizar mapa e selecionar assistência | O usuário deve localizar serviços facilmente |
| **Visualização de Serviços** | Verificar se o usuário consegue visualizar os serviços disponíveis | Selecionar assistência técnica e visualizar serviços | As informações devem ser exibidas de forma clara |
| **Solicitação de Atendimento** | Verificar se o usuário consegue solicitar atendimento | Selecionar serviço e solicitar orçamento | O usuário deve concluir a solicitação sem dificuldades |
| **Painel da Loja** | Verificar se o técnico consegue gerenciar os serviços | Acessar painel, cadastrar serviço e visualizar serviços cadastrados | O técnico deve compreender facilmente o painel |
