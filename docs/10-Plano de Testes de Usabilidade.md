# Plano de Testes de Usabilidade

## 1. Cenários e Tarefas (O que será testado)

| Cenário de Teste                | Objetivo                                            | Tarefas                                                            | Resultado Esperado                                  |
| :------------------------------ | :-------------------------------------------------- | :----------------------------------------------------------------- | :-------------------------------------------------- |
| **Cadastro de Usuário**         | Verificar se o usuário consegue realizar o cadastro  | Acessar tela de cadastro, preencher os dados e finalizar o cadastro | O usuário deve concluir o cadastro sem dificuldades |
| **Login no Sistema**            | Verificar se o usuário consegue acessar o sistema    | Inserir e-mail e senha e selecionar “Entrar”                       | O usuário deve acessar o sistema de forma simples   |
| **Busca de Assistências**       | Verificar se o usuário consegue localizar lojas      | Acessar tela inicial, visualizar mapa e selecionar assistência     | O usuário deve localizar serviços facilmente        |
| **Visualização de Serviços**    | Verificar se o usuário consegue visualizar serviços | Selecionar assistência técnica e visualizar serviços               | As informações devem ser exibidas de forma clara    |
| **Solicitação de Atendimento**  | Verificar se o usuário consegue solicitar serviço    | Selecionar serviço e solicitar orçamento                           | O usuário deve concluir a solicitação sem entraves  |
| **Painel da Loja**              | Verificar se o técnico consegue gerenciar a loja    | Acessar painel, cadastrar serviço e visualizar lista               | O técnico deve compreender facilmente o painel      |
