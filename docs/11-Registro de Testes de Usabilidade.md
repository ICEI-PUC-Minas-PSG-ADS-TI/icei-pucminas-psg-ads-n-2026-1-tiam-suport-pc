# Registro de Testes de Usabilidade

## 1. Visão Geral

O presente documento apresenta os resultados obtidos durante os testes de usabilidade realizados sobre o protótipo da aplicação desenvolvido na ferramenta Figma.

Os testes tiveram como objetivo avaliar a experiência dos usuários durante a navegação pelo sistema, verificando a facilidade de utilização, clareza das informações, organização da interface e compreensão das funcionalidades propostas.

Os participantes executaram tarefas previamente definidas no Plano de Testes de Usabilidade, simulando situações reais de utilização da aplicação.

---

## 2. Participantes

Os testes foram realizados com 5 usuários, representando o público-alvo da aplicação.

Os participantes utilizaram dispositivos móveis para interagir com o protótipo navegável do sistema.

---

## 3. Resultados Obtidos

De forma geral, os participantes conseguiram compreender o funcionamento da aplicação e realizar as tarefas propostas sem grandes dificuldades.

A interface foi considerada simples e intuitiva, principalmente nas funcionalidades relacionadas ao login, navegação entre telas e visualização de serviços.

Durante os testes, também foram identificados alguns pontos de melhoria relacionados à organização visual e ao fluxo de determinadas funcionalidades.

---

## 4. Pontos Positivos Identificados

Os seguintes aspectos foram avaliados positivamente pelos participantes:

- Interface simples e organizada  
- Facilidade de navegação entre as telas  
- Boa identificação das funcionalidades principais  
- Layout visual considerado agradável  
- Facilidade para localizar assistências técnicas  
- Clareza nas informações dos serviços  

---

## 5. Problemas Identificados

Durante os testes, foram observadas algumas dificuldades por parte dos usuários:

- Dificuldade inicial para identificar o fluxo de solicitação de atendimento  
- Falta de destaque visual em alguns botões importantes  
- Necessidade de melhorar a organização de determinadas informações na tela inicial  
- Dúvidas relacionadas à navegação em algumas etapas do sistema  

---

## 6. Melhorias Propostas

Com base nas análises realizadas, foram definidas as seguintes melhorias para futuras versões do protótipo:

- Melhorar o destaque visual dos botões principais  
- Reorganizar informações na tela inicial  
- Simplificar o fluxo de solicitação de atendimento  
- Inserir feedback visual em ações importantes  
- Melhorar a hierarquia visual dos elementos da interface  

---

## 7. Considerações Finais

Os testes de usabilidade permitiram validar o fluxo principal da aplicação e identificar melhorias importantes para a experiência do usuário.

Os resultados demonstraram que o protótipo atende aos objetivos principais do projeto, apresentando uma interface simples, funcional e adequada ao público-alvo definido.
