REGISTRO DE TESTES DE SOFTWARE
1. Visão Geral

Este documento apresenta os resultados dos testes realizados sobre o protótipo da aplicação, desenvolvido na ferramenta Figma.

Os testes tiveram como objetivo avaliar a usabilidade da interface, a clareza das informações e a compreensão do fluxo de navegação pelos usuários.

2. Resultados dos Testes

Os testes foram conduzidos com base em cenários simulados, nos quais os participantes interagiram com o protótipo navegável do sistema.

De forma geral, os usuários conseguiram compreender o funcionamento da aplicação e realizar as tarefas propostas. No entanto, foram identificados pontos de melhoria relacionados à organização das informações e ao fluxo de navegação.

3. Pontos Fortes

Durante os testes, foram identificados os seguintes aspectos positivos:

Interface simples e de fácil compreensão
Navegação intuitiva entre as telas principais
Facilidade na identificação das funcionalidades
Fluxo geral do sistema bem estruturado
Clareza na apresentação dos serviços disponíveis
4. Pontos Fracos

Também foram observadas algumas dificuldades durante a interação com o protótipo:

Dúvidas iniciais sobre o fluxo de agendamento
Falta de destaque em alguns botões importantes
Necessidade de melhor organização das informações em determinadas telas
Ausência de indicações visuais mais claras para orientar o usuário
5. Problemas Identificados

Durante os testes de usabilidade, foram identificados os seguintes problemas:

Dificuldade de alguns usuários em identificar o próximo passo após selecionar um serviço
Falta de feedback visual em ações importantes (como confirmação de agendamento)
Navegação pouco evidente em algumas telas, exigindo tentativa e erro
6. Melhorias Propostas

Com base nos resultados obtidos, foram definidas as seguintes melhorias:

Inserção de feedback visual para ações realizadas pelo usuário
Destaque de botões principais para facilitar a navegação
Reorganização das informações na tela de agendamento
Inclusão de instruções ou elementos visuais que orientem o fluxo de uso
Ajustes na hierarquia visual das telas
7. Considerações Finais

Os testes realizados demonstraram que o protótipo atende aos objetivos propostos, permitindo que os usuários compreendam o funcionamento geral da aplicação.

As melhorias identificadas serão consideradas em etapas futuras do projeto, visando aprimorar a experiência do usuário e a usabilidade do sistema.
